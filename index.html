RabbitFish

a static site generator

RabbitFish is a static site generator written with Python 3. It uses YAML for storing configuration information and content, and Jinja2 for templating. This documentation was generated with RabbitFish.

Features

  • Pages
  • Dynamic Pages

For reference, you can view the RabbitFish Site used to generate this documentation on GitHub: RabbitFish Documentation.

Usage

(From within the directory containing your RabbitFish site and config.yaml)
rabbitfish generatesite
